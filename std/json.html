class JSONValue

This type represents a JSON value.

Example:

auto value = JSONValue([("Hello", JSONValue("World"))]);
assert(value.str == "{\"Hello\": \"World\"}");
assert(JSONValue.parse("{\"Hello\": \"World\"}").str == "{\"Hello\": \"World\"}");
// TODO
// auto value = JSONValue({ "Hello": "World" });
