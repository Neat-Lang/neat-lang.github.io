class JSONValueÔÉÅ

This type represents a JSON value.

Example:

auto value = JSONValue([("Hello", JSONValue("World"))]);
assert(value.toString == "{\"Hello\": \"World\"}");
assert(JSONValue.parse("{\"Hello\": \"World\"}").toString == "{\"Hello\": \"World\"}");
